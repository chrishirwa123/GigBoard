# GigBoard

GigBoard is a modern freelance and local job marketplace built with PHP, MySQL, HTML, CSS, and JavaScript. It connects employers with skilled workers through job posting, applications, private messaging, and profile management.

## Features

- Worker & Employer Authentication
- Role-based Dashboards
- Job Posting
- Browse Jobs
- Job Applications
- Upload CV & Experience Proof
- Private Real-time Messaging
- Worker Profiles
- Employer Profiles
- Application Management
- Responsive UI

## Technologies

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP
